# 统计之都访谈

统计之都访谈是由统计之都发起，由志愿者共同组织的访谈活动。访谈对象主要包括统计学与数据科学相关科教人员、青年学子和产业实践者。无论扬名四海，还是未出茅庐，他们都在努力用数据科学之火点亮千行百业之光。

## 历史访谈文章

在过去8年，统计之都共组织采访或者翻译外文访谈共40多期，绝大多数都进行了编号。
他们之中，有在校学生，有职场新人，有青年教师，有学界业界大佬，也有国内外院士。
